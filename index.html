Connecter Firebase
Pour synchroniser les données en temps réel entre tous les appareils, connectez un projet Firebase gratuit. Opération unique, 5 minutes.
1Créer un projet Firebase
Allez sur console.firebase.google.com → Créer un projet → donnez-lui un nom → désactivez Google Analytics → Créer.
2Activer Realtime Database
Dans le menu gauche → Build → Realtime Database → Create Database → choisissez une région → mode Test → Enable.

Puis dans l'onglet Rules, remplacez par :
{"rules":{".read":true,".write":true}}
Cliquez Publish.
3Récupérer les clés
Roue dentée ⚙️ → Project settings → section Your apps → icône </> (Web) → enregistrez l'app → copiez l'objet firebaseConfig.
4Collez la configuration ici
Collez la valeur de databaseURL
Collez la valeur de apiKey
Collez la valeur de projectId

🎬 Mode démo

Cette instance est une démo publique. Tu peux la réinitialiser à tout moment avec un jeu de données d'exemple (course, points de passage, coureurs, bénévoles), par exemple si un visiteur a modifié ou supprimé des données.
